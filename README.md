# Homologação de Reboque & Renovação CCT INMETRO

Este é um projeto de Landing Page de alta performance focado no nicho de fabricação e regularização de carretinhas e reboques. O objetivo do projeto é unir um design institucional robusto com tecnologias modernas de desenvolvimento e inteligência artificial.

## 🚀 Tecnologias e Inovações

Este projeto foi desenvolvido utilizando ferramentas de última geração:

- **Antigravity IDE**: Utilizado como ambiente de desenvolvimento principal (IDE Agent-First), permitindo uma codificação ágil e inteligente.
- **Sora (OpenAI)**: O vídeo de destaque na primeira dobra (Hero Section) foi gerado via IA Sora, garantindo um visual cinematográfico e exclusivo para o projeto.
- **Design System de Referência**: A interface foi construída baseada em um Design System profissional, utilizando tipografias (Inter e Montserrat) e módulos estruturais de sites de alto padrão como referência estética.
- **Tailwind CSS**: Para uma estilização rápida, responsiva e moderna.
- **GSAP & ScrollTrigger**: Implementação de animações fluidas e interações baseadas no scroll do usuário.
- **Lenis**: Scroll suave (smooth scroll) para melhorar a experiência de navegação (UX).

## 🛠️ Funcionalidades

- [x] Hero section com vídeo em background gerado por IA.
- [x] Design totalmente responsivo (Mobile First).
- [x] Animações de entrada e micro-interações.
- [x] Estrutura otimizada para conversão (Landing Page).
- [x] Integração com Iconify para ícones vetoriais.
